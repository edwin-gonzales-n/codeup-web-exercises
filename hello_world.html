How to add/commit your work to GitHub

  • Create a new file or edit an existing file
  • Any time you add new work, make a commit
  • In IntelliJ, make a commit with Command+K
  • Add a "Commit Message" that explains what you are doing
  • Hover over the "commit" button and select "commit and push"
  • And do it everytime you update something.
  • Practice improve performance

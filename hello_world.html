How to add/commit your work to GitHub

  1. Create a new file or edit an existing file
  2. Any time you add new work, make a commit
  3. In IntelliJ, make a commit with Command+K
  4. Add a "Commit Message" that explains what you are doing
  5. Hover over the "commit" button and select "commit and push"
  6. And do it everytime you update something.
  7. Practice improve performance
